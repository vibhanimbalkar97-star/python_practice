dictionary
A dictionary stores data as key-value pairs.
Keys must be unique and hashable (immutable).
Values can be duplicated and can be of any data type.
Dictionary lookup, insertion, and deletion are O(1) on average because dictionaries use a hash table internally.
Prefer get() when a key may be missing to avoid KeyError.
Use items() when you need both keys and values while iterating.
